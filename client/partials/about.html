About

General

My name is Jerrod Quintana, and this is my first serious project utilizing my MEAN skills, using MonogoDB, ExpressJS, AngularJS, and NodeJS, with a combination of CSS and Bootstap. I wanted to do a message board/forum and what better topic than video games, one of the things I'm most passionate about. You can view the entire website without being logged in but users are unable to do any kind of activity without being logged in.

Users can register, log in, create topics, post on topics and comment on those posts, like posts or comments, and flag inappropriate activity. I utilized both client side and server side validations for all forms and security. You can also see all members, filter different tables, search for the different categories, see info about other users, edit your own profile, and give support or feedback. Admin users can also delete any topics, posts and comments, as well as view all support/feedback tickets. I know there is plenty that can be added onto this site but this is the basic functionality I wanted to implement before I work on my other projects. With any website, new functionality can be added, and problems/bugs can be addressed as need be.
